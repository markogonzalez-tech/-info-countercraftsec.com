# SQL Practice Queries

This repository contains a collection of SQL exercises created while learning database management using MySQL.

## Topics Covered

- SELECT queries
- Aggregation functions (AVG, MAX, MIN)
- GROUP BY and HAVING
- JOIN operations
- Subqueries
- Filtering and sorting data

## Technologies Used

- MySQL
- MySQL Workbench

## Purpose

The goal of this repository is to practice SQL queries and improve database querying skills through different exercises and examples.
